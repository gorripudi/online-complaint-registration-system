# Online Complaint Registration System

## Project Description

Online Complaint Registration System is a full-stack web application developed using Python Flask.  
The system allows users to register, login, submit complaints, and view complaint details easily.

## Technologies Used

- Python
- Flask Framework
- HTML
- CSS
- SQLite Database

## Features

### User Module
- User registration
- User login
- Submit complaints
- View complaint status
- Logout

### Admin Module
- Manage complaints
- Update complaint status
- View registered complaints
